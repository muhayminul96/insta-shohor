The js engine executes JavaScript code. A parser in the js engine, parses JavaScript code. Check the code line by line to see if there are any syntax errors. If there are no errors, Abstract Syntax Trees transform it to machine code. The machine code is subsequently executed by the Processor.

The event loop waits for the Call Stack to be emptied before pushing callbacks from the Task Queue to it.After the Stack is cleaned, the event loop is initiated, and it searches the Task Queue for available callbacks. It moves them to the Call Stack if any are discovered, waits for the Call Stack to clear, and then repeats the operation.
